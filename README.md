# 🌷 Happy Birthday Naura

A romantic birthday website — mobile-first, aesthetic pink & white theme.

## Cara Upload ke GitHub Pages

### Langkah 1 — Buat Repository Baru
1. Buka [github.com](https://github.com) dan login
2. Klik tombol **"New"** atau **"+"** di pojok kanan atas
3. Beri nama repo, contoh: `birthday-naura`
4. Pilih **Public**
5. Klik **"Create repository"**

### Langkah 2 — Upload File
1. Di halaman repo yang baru dibuat, klik **"uploading an existing file"**
2. Drag & drop file `index.html` ke area upload
3. Scroll ke bawah, klik **"Commit changes"**

### Langkah 3 — Aktifkan GitHub Pages
1. Klik tab **Settings** di repo
2. Di menu kiri, klik **Pages**
3. Di bagian **Source**, pilih branch **main** dan folder **/ (root)**
4. Klik **Save**
5. Tunggu 1–2 menit, lalu website bisa diakses di:

```
https://username.github.io/birthday-naura/
```

Ganti `username` dengan username GitHub kamu.
